# ShopNest - E-commerce Marketplace

A complete e-commerce platform built with **HTML, CSS, and JavaScript**. Features buyer and seller dashboards, shopping cart, wishlist, and order management.

## ✨ Features

- **Buyer**: Browse products, filter, search, cart, wishlist, checkout, order tracking
- **Seller**: Add products, manage inventory, view sales and earnings
- **Shopping**: 16+ products across 6 categories
- **Dark/Light Theme**: Toggle anytime
- **Responsive Design**: Works on all devices

## 🚀 Quick Start

1. Open `shopnest.html` in your browser
2. Use demo accounts:
   - **Buyer**: buyer@test.com / buyer123
   - **Seller**: seller@test.com / seller123

## 📊 Dashboard

- **Buyer Dashboard**: Orders, payments, profile
- **Seller Dashboard**: Products, sales, earnings

## 💾 Data

All data stored in browser LocalStorage (persists across sessions)

## 🛠️ Stack

- HTML5, CSS3, Vanilla JavaScript
- No backend or dependencies needed
